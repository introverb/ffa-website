# Example story title

This is the body of an example story. When you port a real Possibilia
piece from Substack, copy the post's content into this file and replace
this placeholder text. Each paragraph is just a paragraph in markdown —
no special wrapping needed.

The first hero image from your Substack post (the one that came embedded
in the story) goes inline here. Markdown image syntax works:

![Caption or alt text](/images/initiative-possibilia.jpg)

Subsequent paragraphs continue the story. Italics work via _underscores_
and **bold** via double asterisks. Links also work — [like this one](https://example.com).

> Block quotes get the sage left-rule treatment and italic body text by
> default — handy for pulling out a striking sentence.

You can also use full headings inside a story when the structure calls
for it.

## A section heading inside the story

Below the heading, more body. Lists work too:

- First thing
- Second thing
- Third thing

When you're happy with how the story renders, save the file — Railway
will rebuild and serve it at `/possibilia/[slug-of-this-folder]`.
