# Example companion essay title

This is the companion essay that pairs with the story. Copy-paste from
the second Substack post (the nonfiction one) into this file.

If a package has no companion piece, just delete this file — the
package page will render with only the story panel and skip the
companion section entirely.

The companion essay author's name comes from `meta.ts`
(`companionAuthor`) and is shown above the body in the eyebrow.
